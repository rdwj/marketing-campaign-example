# Etsy SEO & Keywords Guide

## How Etsy Search Works

Etsy's algorithm considers:
1. **Query matching** - Does your title/tags match the search?
2. **Listing quality** - Views, favorites, conversion rate
3. **Recency** - Newer/renewed listings get a boost
4. **Shipping** - Free shipping ranks higher
5. **Shop quality** - Review score, response time

## Title Structure

Etsy allows 140 characters. Front-load the important stuff.

### Formula
```
[Primary Keyword] [Specific Name] [Condition] [Era] [Secondary Keywords]
```

### Examples

**Premium Item:**
```
Vintage 1997 Beanie Baby Princess Diana Bear Purple PVC Pellets Mint Tags Rare Ty Collectible
```

**Bundle:**
```
Beanie Baby Bundle Lot of 8 Bears Vintage 90s Ty Plush Toys Mint Condition Nostalgic Gift
```

**Bulk Lot:**
```
Beanie Baby Lot 50 Pieces Vintage 90s Ty Wholesale Bulk Destash Craft Supply Reseller
```

## Competitor Research

Before finalizing listings:

1. Search "beanie baby" on Etsy
2. Look at top-selling shops (sort by "Top Reviews")
3. Note their title structures
4. Check their tags (Etsy shows some publicly)
5. Adapt successful patterns

## Common Mistakes to Avoid

1. **Keyword stuffing** - Etsy penalizes unnatural repetition
2. **Wrong categories** - List in Toys > Stuffed Animals
3. **Ignoring attributes** - Fill out all Etsy attributes (color, recipient, occasion)
4. **All caps** - Looks spammy, avoid
5. **Symbols** - Use words, not &, /, etc.

## Renewing for SEO Boost

Etsy gives renewed listings a temporary search boost:
- Costs $0.20 per renewal
- Do this strategically, not constantly
- Best used on slow movers you believe should sell

## Title A/B Testing

For similar items, test different title approaches:

**Version A:** "Vintage Beanie Baby Bear 1997 Ty Plush Mint Tags"
**Version B:** "90s Beanie Baby Teddy Bear Mint Condition Nostalgic Gift"

See which gets more views/favorites over 2 weeks, use winner as template.
