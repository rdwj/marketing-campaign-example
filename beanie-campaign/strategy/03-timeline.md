# 6-Month Campaign Timeline

## Overview

```
Month 1-2: Foundation & Launch
Month 3-4: Ramp-Up & Optimization
Month 5-6: Liquidation & Close-Out
```

---

## Month 1: Foundation

### Week 1: Setup & Research
- [ ] Create Etsy seller account (if needed)
- [ ] Set up shop branding (name, banner, about section)
- [ ] Research your 5 rare pieces on eBay sold listings
- [ ] Order shipping supplies (boxes, tissue paper, tape)
- [ ] Set up photography station (white poster board, natural light)

### Week 2: Premium Inventory
- [ ] Photograph and document all 5 rare/premium pieces
- [ ] Write premium listings using template
- [ ] Research comparable sold prices
- [ ] Publish 5 premium listings

### Week 3: Inventory & Sorting
- [ ] Sort all 1,543 pieces by category (bears, cats, dogs, ocean, etc.)
- [ ] Identify ~50 "desirable but not rare" pieces
- [ ] Note any with condition issues (separate pile)
- [ ] Create spreadsheet tracking inventory

### Week 4: First Bundles
- [ ] Create first 5 themed bundles
- [ ] Photograph bundles (group shots + individual)
- [ ] Write bundle listings
- [ ] Publish bundles
- [ ] Share shop opening on personal social media

**Month 1 Goal:** 10 listings live, shop fully branded

---

## Month 2: Launch Phase

### Week 5: Bundle Production
- [ ] Create 5 more themed bundles
- [ ] List individual "desirable" pieces ($10-25 range)
- [ ] Begin tracking which listings get views/favorites

### Week 6: Optimization
- [ ] Review Etsy stats - what's getting traffic?
- [ ] Adjust titles/tags on underperforming listings
- [ ] Create 5 more bundles
- [ ] Respond to any messages promptly

### Week 7: Marketing Push
- [ ] Set up Instagram or TikTok account for shop (optional)
- [ ] Post first social content
- [ ] Consider starting Etsy Ads on premium items ($1-2/day)

### Week 8: Review & Adjust
- [ ] Calculate sales to date
- [ ] Identify best-performing bundle themes
- [ ] Create more of what's selling
- [ ] Adjust pricing if needed

**Month 2 Goal:** 30 listings live, first sales coming in

---

## Month 3: Ramp-Up

### Week 9-10: Volume Push
- [ ] Create 3-4 bundles per week
- [ ] List remaining "desirable" individual pieces
- [ ] Maintain 50+ active listings

### Week 11-12: Optimization Sprint
- [ ] Refresh listing photos on slow movers
- [ ] Update titles/tags based on Etsy search analytics
- [ ] Run a "Spring Cleaning" sale (10% off bundles)
- [ ] Introduce first bulk lot (20 pieces)

**Month 3 Goal:** 75 listings live, consistent weekly sales

---

## Month 4: Momentum Building

### Week 13-14: Mid-Campaign Review
- [ ] Analyze what's sold vs. what's sitting
- [ ] Double down on successful bundle themes
- [ ] Discontinue underperforming listing styles
- [ ] Create larger bundles (10-12 pieces)

### Week 15-16: Sales Push
- [ ] Run Mother's Day or seasonal promotion (if applicable)
- [ ] "Buy 2 bundles, get free shipping" offer
- [ ] Introduce 50-piece bulk lots
- [ ] Ramp up social media posting

**Month 4 Goal:** 40%+ inventory sold, 100+ listings cycled through

---

## Month 5: Acceleration

### Week 17-18: Markdown Round 1
- [ ] 15% off all listings older than 30 days
- [ ] Larger bundle sizes (15-20 pieces)
- [ ] More aggressive bulk lot offerings
- [ ] Daily social media posts if active

### Week 19-20: Volume Clearing
- [ ] 25% off slow movers
- [ ] "Make an offer" enabled on older listings
- [ ] 100-piece bulk lots introduced
- [ ] Email/message past buyers about bulk deals

**Month 5 Goal:** 65%+ inventory sold

---

## Month 6: Liquidation

### Week 21-22: Final Push
- [ ] 40% off everything
- [ ] Mega bundles (25+ pieces)
- [ ] 250-piece bulk lots at rock-bottom prices
- [ ] "Everything Must Go" shop announcement

### Week 23-24: Close-Out
- [ ] 50% off remaining inventory
- [ ] Combine all remaining into massive lots
- [ ] Consider local consignment or estate sale
- [ ] Calculate donation value for remainder

### Week 25-26: Wrap-Up
- [ ] Final inventory count
- [ ] Donate remainder to charity
- [ ] Document donation for tax purposes
- [ ] Close or pause Etsy shop
- [ ] Celebrate completion

**Month 6 Goal:** 90%+ inventory sold or donated

---

## Weekly Maintenance Tasks

Every week, regardless of phase:

- [ ] Respond to all Etsy messages within 24 hours
- [ ] Ship sold items within 2 business days
- [ ] Renew expiring listings (if keeping)
- [ ] Review Etsy stats dashboard
- [ ] Create at least 2 new listings
- [ ] Post on social media (if active)

## Contingency Plans

### If sales are slow (< 5/week by month 3):
- Drop prices 20% across the board
- Increase bundle sizes
- Boost Etsy Ads budget
- Add listings to eBay as backup channel

### If sales are strong (> 20/week):
- Maintain pricing longer
- Consider hiring help for shipping
- Delay liquidation discounts
- Celebrate

### If a rare piece isn't selling:
- Get it professionally authenticated
- List on eBay simultaneously
- Consider auction format
- Be patient (rare items can take longer)
